Before you worry about folders, version control, or IDEs‚Äîyou need to understand the simplest possible setup.

Why One File?

How to Run It:

1. Ask the AI

The Rapid Iteration Workflow

The real power of AI isn't getting one perfect game‚Äîit's getting something playable, then refining it in seconds.

The Loop:

1. Ask ‚Äî Give the AI a clear prompt with your game idea
2. Receive ‚Äî Copy the complete HTML file
3. Test ‚Äî Open it, play it, notice what feels wrong
4. Refine ‚Äî Tell the AI exactly what to fix
5. Repeat ‚Äî Back to step 2 (usually 3-5 times per game)

üéÆ Real Example: The Mushroom Clicker

Click through the four prompts to see how a game evolves in 5 minutes:

Prompt 1

First Prompt: "Make a clicker game where you click mushrooms to earn points. Include a score counter and a game-over state when you've clicked 50 times."
Result: You get a working game. It works, but the mushrooms don't move around‚Äîthey're in the same spot every time. Boring.

Prompt 3

Next Prompt: "Increase the spawn rate so new mushrooms appear faster. Make the game slightly harder by adding a time limit‚Äîthe player has 30 seconds."
Result: Better. But now you realize there's no feedback when time runs out. One more fix needed.

Prompt 4 - Done!

Final Prompt: "When the time runs out, show a big 'GAME OVER' message with the final score. Add a restart button."
Result: Done. Four prompts. Five minutes. A polished, playable clicker game.

Why This Works:

The Key Insight

You're not asking the AI to rebuild the game each time. You're asking it to fix one specific thing.

This is fast and keeps the AI from breaking what already works.

Surgical fixes. Focused iterations. That's how you go from "nothing" to "polished" in minutes.

Common Pitfalls to Avoid

Here are three things beginners do that waste time. Learn them now, save yourself hours. Click each card to see the better approach:

‚ùå "Rewrite my entire game with this new feature"

‚úÖ "Add a pause button. When the player presses P, the game pauses and shows 'PAUSED'. Press P again to resume."

‚ùå "Make my game better and more fun"
Click to flip
‚úÖ "The game feels too easy. Make enemies spawn twice as fast and move 20% faster."

Why: "Better" is vague. The AI doesn't know if you mean faster, harder, prettier, or more exciting. Be specific.
‚ùå "Fix the bugs"
Click to flip

Why: Describe what's broken in detail. The AI needs the full picture.

The best prompts are specific, surgical, and focused on one problem at a time. Vague = wasted time. Surgical fixes = fast progress.

Getting Assets (Sound, Music, Graphics, Sprites, UI)

A single .HTML file is great for starting, but real games need sounds, music, and visuals. Here's where to get them‚Äîall free.

üéµ Background Music ‚ñº

Where: Suno.ai or similar AI music generators

How: Describe the mood ("upbeat 8-bit chiptune" or "calm ambient music") and let the AI generate it

Format: MP3 or WAV

Tip: Generate 30-60 seconds of looping music. Short loops = small file size

üîä Sound Effects ‚ñº

Where: Freesound.org

How: Search for what you need ("coin sound," "explosion," "click")

Format: WAV or MP3

Tip: Most sound effects are 0.5-3 seconds. Download the shortest one you find

üé® Graphics & Sprites ‚ñº

Where: itch.io/game-assets/free has massive free packs

How: Search for your game style (pixel art, cartoon, minimalist, etc.) and download packs

Format: PNG (best for transparency) or JPG

Tip: Look for "sprite sheets" if you need animated characters

ü§ñ Generated Graphics ‚ñº

Where: ChatGPT, Claude, Bing Image Creator, or similar AI tools

How: Describe what you want ("a happy mushroom character, pixel art style") and download the image

Format: PNG preferred (keeps transparency)

Tip: AI-generated graphics are fast but may need tweaking. Use them as a starting point

üéÆ UI Elements (Buttons, Menus, Borders) ‚ñº

Where: itch.io asset packs OR generate them with AI

How: Search for "UI pack" or ask an AI to generate buttons/menus

Format: PNG

Tip: Consistent UI makes games feel professional. Pick one style and stick with it

Start with free assets. Once your game is polished and you know what you need, you can commission custom art. For now, good-enough assets let you focus on gameplay.

Why This Structure?

How to Set It Up:

Create a new folder called my-game
Inside, create subfolders: sounds, music, images, sprites, ui
Put your index.html file in the root (the main my-game folder)
Drag asset files into their matching folders
In your HTML, reference files like: <img src="images/player.png">

üìå File Size Limits

Why This Structure?

  • Easy to find things ‚Äî Need to change a sound? It's in sounds/
  • Easy to share ‚Äî Zip the entire folder and upload to itch.io
  • Scalable ‚Äî As your game grows, you add folders, not chaos
  • Professional ‚Äî This is how real developers organize games

File Size Limits

Keep your total folder under 100MB. itch.io has soft limits, and larger games take longer to download and play.

  • Music: 3-10MB per track (use MP3 compression)
  • Sound effects: 100KB-1MB each
  • Images: 500KB-2MB each (use PNG compression)

If you're adding video, keep clips under 30 seconds and compress aggressively.
